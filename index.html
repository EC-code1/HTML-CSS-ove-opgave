Webudviklerne

Webudvikler grundforløbet er i gang med forløbet Basics, hvor de lærer om HTML og CSS. Forløbet varer 5 uger.

Når de er færdige med Basics vil webudviklerne have lært følgende:

  • Anvendelse af semantisk HTML til udvikling af websites
  • Anvendelse af semantisk CSS til udvikling af websites
  • Kendskab til tilgængelig webudvikling (accessibility)
  • Kendskab til skalerbart modulær arkitektur i CSS
  • Kendskab til navngivningskonventioner
  • Anvendelse af kode dokumentation
  • Anvendelse af kode editor
  • Anvendelse af browser-indbyggede udviklerværktøj
  • Kendskab til fejlfindingsmetodik i kodning
  • Kendskab til hvordan Internettet fungerer
