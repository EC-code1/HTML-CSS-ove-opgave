Du skal med udgangspunkt i det du har lært sætte siden op med HTML og CSS.

Der er følgende krav til opgaven:

    Løsningen skal være responsiv (indholdet skal tilpasses den skærm det vises på)
    Løsningen skal kodes med en "mobile frist" tilgang.
    Siden "Om holdet" skal indeholde alle elever på holdet (måske kan du finde dem på Microsoft Teams?).
